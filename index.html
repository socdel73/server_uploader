Desti remot

Tria la base de desti, llista subdirectoris i, si cal, afegeix una subcarpeta nova per construir el desti final.

Exemple: /srv/storage-media/uploads_test

Per defecte:

Només es permeten rutes de treball segures. Mai no s’oferiran directoris crítics del servidor.

Llista nomes directoris (nivell 2) dins la base seleccionada.
